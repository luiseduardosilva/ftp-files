# FTP-FILES
> Enviar arquivos via FTP.

Script criado para enviar de arquivos via FTP.

## Config.ini

\# | Nome  | Descrição
------------- | ------------- | -------------
:house: | host  | Host FTP
:cop: | username  | Usuário FTP
:key: | password  | Senha FTP
:floppy_disk: | dir	 | Diretorio Local
:computer: | remoto | Diretorio FTP



## Exemplo

	[Teste]
	host = 127.0.0.1
	username = luissilva
	password = admin321
	dir = C:\FTP
	remoto = /pasta/
